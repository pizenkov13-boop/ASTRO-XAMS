# Ad-lib audio

Drop **any number** of MP3 clips here. The app loads them automatically — no code changes.

## Naming

```
adlib (1).mp3
adlib (2).mp3
…
adlib (11).mp3
```

Other audio names work too (`.mp3`, `.wav`, `.ogg`, `.m4a`). Files are sorted by the number in `adlib (N)` when present.

## Behavior

- **Every correct answer** → random clip from this folder
- **5 correct in a row** → another random clip at full volume + screen celebration

Restart `npm run dev` after adding new files.

## Legal

Use only audio you have the right to use.
