# Vision board images

Drop **any number** of photos here. They are loaded automatically — no code changes needed.

## Naming

Works great with numbered files:

- `img (1).jpg`
- `img (2).jpg`
- `img (3).jpg`
- …

Other image names are supported too (`.jpg`, `.jpeg`, `.png`, `.webp`, `.gif`, `.avif`). Files are sorted by the number in `img (N)` when present.

## After adding files

Restart the dev server if images do not appear immediately:

```bash
npm run dev
```

Then open **VISION** in the app nav.
